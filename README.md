Overview

This is a smart chat application built using HTML, CSS, and JavaScript that simulates AI-like conversation.

The app provides an interactive chat experience with dynamic responses and real-time UI updates.

---

🚀 Features

- 💬 Send and receive messages
- 🤖 AI-like smart replies
- ⏳ Typing simulation ("AI is typing...")
- 💾 LocalStorage (messages saved after refresh)
- 🧹 Delete selected messages
- 🗑 Clear entire chat
- ⌨️ Enter key support

---

🛠 Tech Stack

- HTML
- CSS
- JavaScript
